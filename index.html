Project Overview

This portfolio focuses on learning MySQL through the creation and management of a product database. It covers fundamental tasks such as creating tables, setting constraints, inserting accurate data, and modifying fields. The aim is to gain a clear understanding of how to structure and manage a database using basic SQL commands.

Download Document

Step by Step Process

  • Create the Table: Create a `products` table with:
    • `id` (auto-increment, primary key)
    • `product_name` (VARCHAR(100), NOT NULL)
    • `price` (DECIMAL)
  • Add a Rule: Add a CHECK constraint to ensure `price > 0`.
  • Insert Valid Products: Insert:
    • Laptop – 999.99
    • Smartphone – 599.99
    • Tablet – 299.99
    • Keyboard – 19.99
    • Mouse – 14.99
    • Desk Lamp – 24.99
    • Speakers – 9.99
  • Update the Table: Alter `product_name` to support up to 120 characters.
